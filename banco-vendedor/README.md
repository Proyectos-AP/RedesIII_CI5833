# Banco del vendedor

## Instalación

## Crear un ambiente virtual con `venv`
```bash
python3 -m venv env
source env/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

## Crear la BD.
```bash
python bdBancoVendedor.py
```

## Popular la BD.
```bash
python add.py
```

# Ejecución
```bash
python server.py
```
